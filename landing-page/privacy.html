Tabi AI Privacy Policy

Last updated: October 2025

Tabi AI is a privacy-first browser assistant that helps users interact with Chrome using natural language. It allows you to search, organize, and manage tabs and bookmarks intelligently — all without sending your browsing data to external servers.

1. Data Collection and Processing

Tabi AI does not collect, transmit, or store any personal data, browsing history, or website content. All processing happens locally on your device using Chrome’s built-in AI APIs (chrome.ai.languageModel and chrome.ai.writer).

The extension temporarily accesses:

  • Tab metadata (title, URL) to perform local semantic search and organization.
  • Bookmark metadata to organize, search, or remove bookmarks upon user request.

This data is never transmitted, logged, or shared externally. Once a task is complete, it is discarded from memory.

2. Permissions Explained

Tabi requests certain Chrome permissions solely to provide its single purpose — intelligent local browser assistance:

  • tabs / tabGroups / windows: To view and organize open tabs.
  • bookmarks: To search and modify user bookmarks on request.
  • scripting: To run lightweight local scripts connecting to Chrome’s AI APIs.
  • storage: To save minimal settings like theme or interface state.
  • host_permissions ("<all_urls>"): Required to access tab titles and URLs for AI reasoning across any site — but no page content is read or modified.

3. Remote Code and External Services

Tabi AI does not use any remote code, external JavaScript, or third-party analytics. All scripts are bundled within the extension package and reviewed by Chrome Web Store policies.

4. Data Sharing and Sale

Tabi AI does not sell, share, or transfer user data to any third party. No data is used for advertising, analytics, or profiling.

5. Transparency and Control

All AI actions are visible to the user. When a task is processed locally, it is executed by Chrome’s on-device Gemini Nano model. For rare fallback cases that use the Gemini 2.5 Flash backend, only non-personal task context is sent and not logged.

6. Contact

For privacy-related questions or feedback, contact the developer at: contact@tabiai.dev
